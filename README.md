## 🔭 the flagship project
### **Commute** 🚦
*A traffic flow prediction system using spatio-temporal modeling*

- **Model:** Spatio-Temporal Graph Neural Network  
- **Architecture:**  
  - Graph Attention Networks (GAT) for spatial dependencies  
  - Mamba (State Space Models) for temporal modeling  
- **Goal:** Accurate traffic forecasting by capturing complex spatial + temporal patterns  
- **Status:** Actively developing 🚧
